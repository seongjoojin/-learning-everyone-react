# 프론트엔드 라이브러리

생산성과 유지보수성이 좋음

프론트엔드 라이브러리 3대장

1. 앵귤러 (Angular)

다양한 기능이 내장되어 있음. 훌륭한 프레임워크

2. 리액트 (React)

컴포넌트에 집중된 라이브러리. 한가지 방법을 해결하기 위해 많은 방법이 있음.

3. 뷰 (Vue)

입문자가 사용하기에 좋은 라이브러리.
